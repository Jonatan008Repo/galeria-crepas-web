# Galería de Crepas — Website

Sitio web oficial del restaurante **Galería de Crepas**, ubicado en Puebla, México.

El proyecto busca fortalecer la presencia digital del restaurante mediante una experiencia web visual, rápida y optimizada para dispositivos móviles, permitiendo a los clientes explorar el menú, descubrir el concepto artístico del lugar y acceder fácilmente a la ubicación y redes sociales.

---

# Descripción del Proyecto

**Galería de Crepas** es un restaurante con un concepto que combina gastronomía y arte.

El establecimiento ofrece:

* crepas dulces y saladas
* crepas especiales
* un sistema de menú simple y personalizable
* una galería con obras de artistas locales

Este sitio web tiene como objetivo comunicar esa experiencia de manera clara y visual.

---

# Objetivos del Sitio

1. Mostrar el **menú digital** del restaurante.
2. Facilitar el acceso al **menú mediante códigos QR en las mesas**.
3. Reflejar el concepto artístico del restaurante.
4. Integrar redes sociales como **Instagram** y **WhatsApp Business**.
5. Mejorar la visibilidad del restaurante en **búsquedas locales**.
6. Permitir que el menú pueda **actualizarse fácilmente**.

---

# Características del Sitio

* diseño **mobile-first**
* menú optimizado para **escaneo QR**
* experiencia visual basada en fotografía
* integración con **Google Maps**
* acceso rápido a **Instagram y WhatsApp**
* estructura preparada para futuras funciones de pedidos

# Flujo de Usuario

Escenario típico:

1. El usuario descubre el restaurante en Instagram.
2. Visita el sitio web.
3. Explora la galería y el menú.
4. Consulta la ubicación.
5. Visita el restaurante.

Segundo escenario:

1. Cliente dentro del restaurante.
2. Escanea QR en la mesa.
3. Consulta el menú digital.
